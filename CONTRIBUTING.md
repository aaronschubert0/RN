Remember to adhere to the code style:

```jsx
// one prop
<Component prop={value} />
```
```jsx
// one prop + children
<Component prop={value}>
  {children}
</Component>
```
```jsx
// multi props + children
<Component 
  foo={bar}
  baz={bax}
>
  {children}
</Component>
```
```jsx
// multi props + no children
<Component 
  foo={bar}
  baz={bax}
/>
```
